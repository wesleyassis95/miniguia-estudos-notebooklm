Resumos Estruturados:
- Definição: Machine Learning é um campo da IA que permite que sistemas aprendam padrões a partir de dados sem serem explicitamente programados.

Tipos de Aprendizado:
- Supervisionado (usa dados rotulados, ex: classificação de spam).
- Não supervisionado (descobre padrões sem rótulos, ex: segmentação de clientes).
- Reforço (aprende por tentativa e erro, ex: jogos e robótica).

Principais Algoritmos: Regressão Linear, Árvores de Decisão, K-Means, Redes Neurais.Aplicações: Reconhecimento de imagens, recomendação de produtos, diagnósticos médicos, previsão de demanda.

Glossário:
- Dataset: Conjunto de dados usado para treinar modelos.
- Overfitting: Quando o modelo aprende demais os detalhes do treino e perde capacidade de generalização.
- Feature: Variável usada como entrada para o modelo.
- Modelo: Representação matemática treinada para fazer previsões.
